# Dotfiles for macOS

I use [gnu stow](https://www.gnu.org/software/stow/) to manage my dotfiles.

I typically use nu-shell, but I have a config for zsh too.

Editor is [lunarvim](https://www.lunarvim.org/) and prompt is [starship](https://starship.rs/).

Color scheme is [Tokyo Night](https://github.com/folke/tokyonight.nvim)

Font is `FiraCode Nerd Font Mono`
